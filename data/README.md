# data/

Raw and processed data for the Influencer Marketing (Essay 2) project.

- **Not committed.** Data contents are gitignored (this README is the exception). The
  working data lives on the author's machine (`Dropbox/.../Revised Version JM/Derived`).
- **Confidential handling:** the `.claude/rules/confidential-data.md` rule guards `data/**`.
  Treat influencer-level records as restricted; do not paste raw rows into chat, commits,
  or outputs.
- Suggested layout: `data/raw/` (as received, read-only) and `data/derived/` (cleaned,
  script-generated).
